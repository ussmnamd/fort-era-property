## 3. SECURITY REQUIREMENTS

Apply all of the following. No exceptions.

- **All API keys, DB credentials, and secrets** in `.env` / `.env.local` вЂ” never committed to git
- Add `.env*` to `.gitignore` immediately
- Provide `.env.example` with all variable names and placeholder values
- **Rate limiting** on all form submission endpoints (use `express-rate-limit` in Strapi or middleware in Next.js API routes)
- **CORS** locked to your domain in Strapi config вЂ” no wildcard in production
- **Input sanitization** on all form fields (server-side, not just client-side)
- **CSRF protection** on all POST routes
- Strapi API tokens scoped to minimum permissions (read-only token for frontend, write token for lead capture only)
- **Helmet.js** enabled in Strapi server config
- No sensitive data (phone numbers, email addresses) exposed in public API responses вЂ” use Strapi field-level permissions
- WhatsApp phone number stored in env variable, never hardcoded in source
- All image uploads validated by type and size in Strapi (images only, max 10MB)

## 9. API LAYER (`/lib/api.ts`)

```typescript
const STRAPI_URL = process.env.NEXT_PUBLIC_STRAPI_URL;
const STRAPI_TOKEN = process.env.STRAPI_READ_TOKEN;

export async function fetchAPI<T>(path: string, params?: Record<string, string>): Promise<T> {
  const query = params ? '?' + new URLSearchParams(params).toString() : '';
  const res = await fetch(`${STRAPI_URL}/api${path}${query}`, {
    headers: { Authorization: `Bearer ${STRAPI_TOKEN}` },
    next: { revalidate: 60 },
  });
  if (!res.ok) throw new Error(`Strapi fetch failed: ${path}`);
  return res.json();
}
```

Use `next: { revalidate: 60 }` for ISR on listings. Use `cache: 'no-store'` for lead submission.

## 10. LEAD CAPTURE & EMAIL

On form submission:
1. POST to Next.js API route `/api/leads`
2. Validate + sanitize with `zod`
3. POST to Strapi `/api/leads` with write token
4. Send email via Nodemailer to agent's email (from env)
5. Return JSON response

Email includes: name, phone, email, inquiry type, property reference, message.
